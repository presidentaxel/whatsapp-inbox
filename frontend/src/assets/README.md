# Assets

Ce dossier contient les ressources graphiques de l'application.

## Logo LMDCVTC

Pour le moment, le favicon SVG de `public/favicon.svg` est utilisé dans l'état vide du chat.

### Pour ajouter un logo plus grand :

1. Placez votre logo dans ce dossier (formats recommandés : PNG, SVG, JPG)
2. Nommez-le `logo-lmdcvtc.png` ou `logo-lmdcvtc.svg`
3. Le composant `ChatWindow.jsx` l'importera automatiquement

**Recommandations :**
- Dimensions recommandées : 200x200px minimum
- Format transparent (PNG ou SVG) préféré
- Poids < 100KB pour de bonnes performances
