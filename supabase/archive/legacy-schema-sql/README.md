# Anciens scripts `supabase/schema/*.sql`

Ces fichiers sont conservés **à titre historique** (référence, comparaison, audit). Ils ne sont **plus** exécutés par le CLI Supabase : tout le schéma initial pertinent a été intégré dans `supabase/migrations/001_baseline_legacy_schema.sql`, puis les évolutions suivent les migrations numérotées dans `supabase/migrations/`.

Ne pas les ré-appliquer sur une base déjà migrée sans savoir ce que vous faites (risque de conflit avec les migrations).
